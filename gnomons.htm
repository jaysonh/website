Gnomons

Gnomons is an interactive light sculpture which was exhibited at the outdoor festival Shine On in regional Victoria in November 2011. The sculpture allows viewers to interact with the images and patterns being projected onto the sculpture by moving there hands inside of a hollow frame.

Gnomons was created using a projector to project onto 18 solid wooden posts, software which was custom built using OpenFrameworks was used to create images and patterns to project onto the posts and an Xbox kinect was also used to do hand tracking to allow people to interact with the sculpture.

This installation was presented at:

Shine On Festival, Australia 2011
Entheogenesis Australia 2011
